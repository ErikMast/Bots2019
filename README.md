<p>Laravel 6 Bots</p>
<h2>Quick Start</h2>

<p><strong>If never used laravel before run this</strong><br />
composer global require "laravel/installer"

<p><strong>.env</strong><br />
change the content of the .env to the right information and change the file to .env</p>

<p><strong>Install Dependencies</strong><br />
composer install</p>

<p><strong>Run Migrations</strong><br />
php artisan migrate</p>

<p><strong>Add virtual host if using Apache</strong><br />

<p><strong>If you get a error about an encryption key</strong><br />
php artisan key:generate</p>
